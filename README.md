# h_dawei_cli
![Npm 版本](https://img.shields.io/badge/h_dawei_cli_v0.0.1-green)
## 📖简介
- 用于快速搭建前端项目的命令行工具

## 📕模版
- ✅快速生成[Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模版](https://github.com/Dreeze549/h_dawei_cli)

## 📦安装
``` bash
npm install h_dawei_cli -g
```

## 💯使用
``` bash
#使用模版创建新项目
dawei create

# 执行项目文件夹名称并选择模版创建新项目
dawei create [projectName]

# 更新脚手架到最新版本
dawei update

#查看脚手架版本
dawei -v
dawei -version
```
